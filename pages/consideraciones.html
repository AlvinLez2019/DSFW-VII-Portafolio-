Consideraciones

[Inicio de Pág.]

Durante estas semanas se pudo notar que aprender programación web requiere práctica constante, ya que al inicio pueden surgir varios errores, especialmente con la configuración de herramientas como XAMPP o el uso de PHP. También es importante tener bien organizados los archivos y rutas para evitar problemas al ejecutar los proyectos.

Otra cosa que considerar es que entender la lógica (como condicionales y ciclos) es clave para poder avanzar, ya que eso permite resolver mejor los ejercicios. Además, el uso de herramientas como GitHub ayuda a llevar un mejor control del trabajo.

En general, es importante tener paciencia y practicar, porque, aunque al inicio cueste, con el tiempo todo se va entendiendo mejor.
