Este portafolio presenta mi trayectoria en el curso de desarrollo de software VII. Este curso tuvo como objetivos desarrollar las destrezas de aprendizaje en el lenguaje php y HTML para ser usados en el servidor xampp; además, adquirí las siguientes competencias: creación de pagina web, logica del lenguaje php y html.

Durante este semestre pude conocer acerca de: las diferencias que hay entre el lenguaje de programación C++ y php.

La técnicas e información mostradas por el profesor me ayudaron a mejorar como futuro profesional de la informática y las tecnologías.

Las áreas más difíciles del curso fueron: la organización del código e implementar html al código php y que el algoritmo permita ingresar informacion desde la barra de busqueda.
